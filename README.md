# Exotikmat — Vite Project

This project has been migrated to a modern Vite + React + Tailwind CSS setup.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Project Structure

- `src/` — Source code
  - `components/` — React components (Home, Checkout, etc.)
  - `App.jsx` — Router and main layout
  - `main.jsx` — Entry point
  - `index.css` — Tailwind v4 CSS and custom styles
- `index.html` — Vite template
- `vite.config.js` — Vite configuration
- `tailwind.config.js` — Tailwind configuration
- `index.legacy.html` — Backup of the original single-file index

## Key Technologies

- **React 18**
- **Vite 5**
- **Tailwind CSS 4** (using `@tailwindcss/vite` plugin)
- **Lucide React** for icons
- **React Router 6** for navigation
